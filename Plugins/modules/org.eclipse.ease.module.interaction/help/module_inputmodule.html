Module InputModule

This module help to ask the user informations

Function Overview

askSimpleString Opens a simple dialog box to ask the user a simple string.
ask Opens a simple dialog box to ask the user a simple string with an empty default value and default dialog title.
askConfirmation Opens a dialog box to ask a confirmation to the user (e.g. before a destructive or expensive operation).
askQuestion Opens a dialog box to ask a simple Yes/No question to the user.
selectContainer Opens a dialog box which allows the user to select a container (project or folder) in the workspace.
selectResource Opens a dialog but which allows the user to select a resource from the workspace.
selectResource Opens a dialog but which allows the user to select a resource from the workspace with a default title and message.
readFile Reads the contents of a text file from the workspace into a string.
readFile Reads the contents of a text file into a string.
readFile Reads the contents of a text file into a string.
